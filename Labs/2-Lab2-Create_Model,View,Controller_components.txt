

1- Models klasörü altına Urun sınıfı oluşturma.
 Model Klasörü altına Urun.cs adında bir sınıf oluşturunuz.

2-Aşağıda verilen property'leri tanımlayınız. (Kısayol prop + tab + tab)
    string tipinde Id;  // 1
    string tipinde Isım;  // Lenovo Ultrabook V4
    string tipinde Aciklama;   // Bu ürünün ağırlığı yaklaşık 2400 gramdır.
    string tipinde Kategori;   // Elektronik
    int tipinde Fiyat;  // 13999

3- UrunController.cs adında bir Controller oluşturunuz.
 Oluşturduğunuz Controller sınıfı altında Index Metodunda Urun sınıfından bir nesne oluşturunuz.
 Oluşturduğunuz nesnenin özelliklerine istediğiniz bir ürünün özelliklerini atayınız.

4- Oluşturduğunuz nesnenin örneğini UrunController.cs sınıfında bulunan Index metodundan Views/Urun/Index.cshtml adındaki View dosyasına gönderiniz.

5- View Klasörünün altında bulunan Urun klasöründe bulunan Index.cshtml dosyasının içinde oluşturduğunuz Urun sınıfı nesnesinin her bir özelliğinin değerlerini gösteriniz!

6. Oluşturuduğunuz Urun View sayfasına erişim linkini  Shared >_Layout.cshtml  altına ekleyiniz.


7- Yaptığınız çalışmaları kendi GitHub hesabınızda oluşturuduğunuz web tabanlı uygulama geliştirme için kullandığınız repositorinize ekleyiniz.
